# 单体架构Wiki示例

本文档提供单体应用项目的Wiki目录结构和文档模板。

---

## 推荐目录结构

```
docs/
├── README.md                          # Wiki首页/项目简介
├── 01-技术架构/
│   ├── 01-技术栈说明.md               # 技术栈详细说明
│   ├── 02-系统架构说明.md             # 系统分层架构
│   ├── 03-业务架构说明.md             # 业务域划分【重点】
│   ├── 04-功能架构说明.md             # 功能模块结构
│   ├── 05-目录结构说明.md             # 项目目录结构详解
│   └── 06-核心流程说明.md             # 核心业务流程【新增】
├── 02-业务模块分析/
│   ├── 01-[模块A].md                  # 包含流程图、时序图、核心流程
│   ├── 02-[模块B].md                
│   └── ...
├── 03-数据库设计/
│   ├── 01-数据库概述.md               # 数据库整体设计
│   ├── 02-ER图总览.md                 # 模块间核心表关联【新增】
│   ├── 03-[模块A]ER图.md              # 按模块划分的详细ER图【新增】
│   ├── 04-[模块B]ER图.md              # 按模块划分的详细ER图【新增】
│   ├── 05-表索引.md                   # 所有表清单（确保无遗漏）【新增】
│   └── 06-表结构详情.md               # 各表结构说明
├── 04-开发规范/
│   ├── 01-代码规范.md
│   ├── 02-命名规范.md
│   ├── 03-分层规范.md
│   └── 04-API规范.md
├── 05-部署运维/
│   ├── 01-本地开发环境.md
│   ├── 02-部署方式.md
│   └── 03-配置说明.md
└── 06-附录/
    ├── 01-术语表.md                   【重点】
    └── 02-常见问题.md
```

---

```markdown

> [一句话项目简介]

---

[2-3段项目介绍]

| 类别 | 技术 | 版本 |
|-----|------|------|
| 框架 | Spring Boot | 2.x |
| ORM | MyBatis-Plus | 3.x |
| 数据库 | MySQL | 5.x |

## 核心功能

[Mermaid mindmap展示核心功能]

## 文档导航

- [技术架构](01-技术架构/)
- [业务模块分析](02-业务模块分析/)
- [数据库设计](03-数据库设计/)
- [开发规范](04-开发规范/)
- [部署运维](05-部署运维/)
- [附录](06-附录/)

## 快速开始

### 环境要求
- JDK 1.8+
- Maven 3.x
- MySQL 5.x

### 启动命令
mvn spring-boot:run
```
