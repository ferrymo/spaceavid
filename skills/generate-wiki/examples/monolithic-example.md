# 单体架构Wiki示例

本文档提供单体应用项目的Wiki目录结构和文档模板。

---

## 推荐目录结构

```
docs/
├── README.md                          # Wiki首页/项目简介
├── 01-技术架构/
│   ├── 01-技术栈说明.md               # 技术栈详细说明
│   ├── 02-系统架构说明.md             # 系统分层架构
│   ├── 03-业务架构说明.md             # 业务域划分【重点】
│   ├── 04-功能架构说明.md             # 功能模块结构
│   └── 05-目录结构说明.md             # 项目目录结构详解
├── 02-业务模块分析/
│   ├── 01-[模块A].md                  # 包含流程图、时序图
│   ├── 02-[模块B].md                
│   └── ...
├── 03-数据库设计/
│   ├── 01-数据库概述.md               # 数据库整体设计
│   ├── 02-核心表ER图.md               # ER图
│   └── 03-表结构详情.md               # 各表结构说明
├── 04-开发规范/
│   ├── 01-代码规范.md
│   ├── 02-命名规范.md
│   ├── 03-分层规范.md
│   └── 04-API规范.md
├── 05-部署运维/
│   ├── 01-本地开发环境.md
│   ├── 02-部署方式.md
│   └── 03-配置说明.md
└── 06-附录/
    ├── 01-术语表.md                   【重点】
    └── 02-常见问题.md
```

---

```markdown
## 快速开始

### 环境要求
- JDK 1.8+
- Maven 3.x
- MySQL 5.x

### 启动命令
mvn spring-boot:run
```
